================== ML GNIP Connector ==================
MarkLogic Connector to wrap GNIP stream 
1. Accept GNIP stream in gzip format
2. Extract json from gzip
3. JSON to XML transformation
4. Enrich data using cts:highlight and element co-occurences
5. Ingest data into ML
		** copy the lib folder into indiviual folders prior to running the script **

Usage:
$	sh twitter-start.sh
